Mission Brief

Welcome to Symbol Table City! Complete the symbol table by identifying properties of variables in the source code.

Symbol Table — a compiler data structure that records information about every identifier.
Each entry tracks:
• Identifier — variable name
• Type — data type (int, float, etc.)
• Value — assigned value (or —)
• Scope — global or local
• Line — line of declaration
Scope Rules:
• Variables declared outside { } → global
• Variables declared inside { } → local
