Mission Brief

Welcome to the FIRST & FOLLOW Bridge! Compute the FIRST and FOLLOW sets for every non-terminal in the grammar.

FIRST(X) — the set of terminals that can begin strings derived from X.
• If X → a..., add a to FIRST(X)
• If X → Y..., add FIRST(Y) to FIRST(X)
• If X can derive ε, add ε to FIRST(X)
FOLLOW(X) — the set of terminals that can appear immediately after X.
• Add $ to FOLLOW(start symbol)
• If A → αXβ, add FIRST(β)−{ε} to FOLLOW(X)
• If A → αX or ε ∈ FIRST(β), add FOLLOW(A) to FOLLOW(X)
Input format: Comma-separated symbols. Example: a, b
Use ε for epsilon and $ for end-of-input.

You have 3 attempts. After 3 failures, the correct answers will be shown.
