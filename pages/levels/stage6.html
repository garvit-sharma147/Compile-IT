Key concepts:

Non-terminals — Uppercase symbols (S, A, B) that expand into other symbols.
Terminals — Lowercase characters (a, b, c) that appear in the final string.
Start Symbol — The first non-terminal (left side of the first rule).
ε (epsilon) — Represents the empty string.

Valid strings can be produced by repeatedly applying grammar rules starting from the start symbol.
